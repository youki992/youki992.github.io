团队介绍

Description

我们是一群对网络安全充满热情的年轻探险者，Code4th安全团队（简称C4安全）正式成立于2024年7月。

团队热衷于网络安全，致力于挖掘国内外SRC平台漏洞。

我们的座右铭是："成败在于细节"。梦想是成为专业的漏洞挖掘团队，共同探索网络安全的无限可能~

开发工具

Developed hack tools

VscanPlus

Vscan综合扫描工具的二次开发版本，基于veo师傅的漏扫工具vscan二次开发的版本，端口扫描、指纹检测、目录fuzz、漏洞扫描功能工具，批量快速检测网站安全隐患。

工具地址

Fofa-Cloud-Server

基于单FOFA账号部署的多人协作查询B/S模式，后端部署后通过前端网页交互查询数据和导出数据，前端由html开发，后端由spring-boot开发。根据不同情况可以部署在本地局域网中，或是部署在服务器上，使用Apache或Nginx进行配置前端页面。

工具地址

Goby插件 - VscanPlus

Goby联动VscanPlus的插件，可对单个IP进行更加深度的漏洞、指纹、端口扫描检测。可在Goby官方插件商店中下载。

工具地址

代码审计辅助工具 - CodeStringSearcher

基于Java开发的代码审计工具，用于辅助快速代码审计，筛选危险方法名称搜索代码中可能存在的漏洞。config.properties配置文件中的searchText可以配置多个字符串关键字，并用英文逗号分割以方便搜索。
